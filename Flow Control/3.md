# Code Block
## 程式區塊
在開始流程控制之前，還有一個程式區塊的概念需要暸解。
Python語法當中，是使用空白鍵或者是tab來區分同一個程式區塊。

```python
myName = 'Jason'
password = '123456'
print(myName + '，請輸入您的密碼：')
inputPassword = input()
if password == inputPassword:
    print('welcome')
    print('your password is correct')
else:
    print('incorrect password')
```

> myName = 'Jason'  
> ...  
> else:

以上在同一個縮排等級的為同一個block，
